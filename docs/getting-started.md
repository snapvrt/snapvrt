## Prerequisites

- Node.js 18+
- Docker
- Storybook 10+

## Installation

```bash
npm install -D snapvrt
```

## Initialize

```bash
snapvrt init
```

## Capture References

```bash
snapvrt update
```

## Run Tests

```bash
snapvrt test
```

## Review Changes

```bash
snapvrt review
```
